# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## Paleta de Cores (Site)

Atualmente, o site utiliza as seguintes cores, definidas em `src/index.css`:

- **Primária (Verde Sage):** `#8A9A86` (Usada em botões, títulos e favicon)
- **Primária Hover:** `#6B7B68` (Versão mais escura para efeitos de hover)
- **Primária Light:** `#E4E9E2` (Verde muito claro para fundos sutis e badges)
- **Accent (Bege/Dourado):** `#D4C4A8` (Usada para detalhes e interações)
- **Fundo (Off-White):** `#FAF9F6`
- **Texto:** `#2C3329` (Escuro) / `#5C6658` (Suave)

Para alterar a identidade visual, basta atualizar os valores das variáveis CSS em `:root` no arquivo `src/index.css`.
